## 螺絲孔 (Mounting Hole)

```
⌀ 3mm (螺絲通過孔徑)
⌀ 5mm (銅箔環, 連 GND)
4 個角落各 1 個, 距板邊 3mm

建議：
  - 不使用金屬螺絲直接接地（避免天線效應）
  - 使用塑膠螺絲 + 銅柱固定於 3D 列印外殼
```

## 佈線注意事項

### 電源走線

```
           ← 最大電流 →     ← 建議線寬 →
VBUS (5V)    500mA            0.5mm  ~ 1.0mm
BAT (3.7V)   500mA            0.5mm  ~ 1.0mm
3.3V         300mA            0.5mm
GND          return           大面積鋪銅

via stitching：電源每 5mm 至少一對 via 連通頂底層
```

### 訊號走線

```
I2C (SDA/SCL)  ← 0.3mm, 遠離天線, 等長
ADC (BAT_DIV)  ← 0.3mm, 遠離數位訊號
USB D+/D-      ← 0.3mm, 差分對, 等長 < 0.5mm
GPIO           ← 0.3mm

I2C 走線長度：SDA = SCL, 誤差 < 2mm
USB 差分阻抗：90Ω (若長度 > 10mm, 需計算微帶線寬)
```

### 阻抗控制 (USB)

```
若 USB D+/D- 走線長度 > 10mm：

  FR-4 (εr = 4.3), 1.6mm, 1oz
  微帶線邊緣耦合差分對：
    Trace width:  0.35mm
    Trace space:  0.15mm
    Ground clearance below: 整層 GND
  目標差分阻抗: 90Ω ± 10%

(若走線 < 10mm, 直接用 0.3mm 即可)
```
