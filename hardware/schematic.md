## 詳細電路

### 1. 電源電路（內部 LiPo + 外部 14500 雙電源，自動 OR-ing）

```
                          USB 5V
                            │
                       [IP2362A]
                            │
                   ┌────────┴────────────┐
                   │  BAT (充電輸出)      │
                   │      │              │
                   │ 內部 JST1.25        │
                   │ LiPo 402030 200mAh  │
                   │      │              │
                   │  ┌───┴────┐         │
                   │  │  D1    │         │
                   │  │ SS12   │ 0.2V 壓降
                   │  └───┬────┘         │
                   │      │              │
                   │   ┌──┴──┐           │
                   │   │    ├──── BAT_ADC (GPIO0 分壓)
                   │   │    │            │
                   │   │  ┌─┴────────┐   │
                   │   │  │ ME6211   │   │
                   │   │  │ 3.3V LDO │   │
                   │   │  └──┬───────┘   │
                   │   │     │           │
                   │   │  3.3V rail      │
                   │   │     │           │
                   │   │  10µF  0.1µF   │
                   │   │     │      │    │
                   │   │    GND    GND   │
                   │   │                │
                   │      EXTERNAL_BAT  │
                   │   JST 2.0 / 端子    │
                   │   ┌────────┐       │
                   │   │  D2    │       │
                   │   │ SS12   │ ← 14500 / 16340
                   │   └───┬────┘        │
                   │       │             │
                   └───────┴─────────────┘

D1, D2 = SS12 Schottky (1A, 20V, VF ≈ 0.2V @ 100mA)
```

**自動選擇邏輯**：

| 狀況 | 內部 LiPo (402030) | 外部 14500 | 電源來源 |
|------|-------------------|-----------|---------|
| 僅內接 | 3.7~4.2V | 無 | 內部 LiPo |
| 僅外接 | 無 | 3.7~4.2V | 外部 14500 |
| 兩者皆有 | 3.7~4.2V | **4.2V (剛充飽)** | **外部**（電壓高者勝出）|
| 皆無 | 0V | 0V | 斷電 |

**優點**：
- 隨插即用，自動切換，不需開關或跳線
- 可熱插拔外部電池（由 D2 防止回灌）
- 外部 14500 可更換，場域測試不中斷

**缺點**：
- Schottky 壓降 0.2V → LDO 實際輸入降低約 3.0V（但仍高於壓差需求）
- ADC 監測的是 OR-ing 後電壓，需軟體判別電池來源

### 2. I2C 匯流排

```
+3.3V
  │
  ├─── 4.7kΩ ────┬──── SDA (ESP32-C3 GPIO6)
  │               │
  │               └──── SDA (MPU6050 pin 24)
  │
  ├─── 4.7kΩ ────┬──── SCL (ESP32-C3 GPIO7)
  │               │
  │               └──── SCL (MPU6050 pin 23)
  │
  └─────────────────── VCC (MPU6050 pin 8)
                      AD0 (MPU6050 pin 9) ──► GND

MPU6050 其餘腳位：
  INT (pin 12)  ──► NC (保留)
  CLKIN (pin 13) ──► GND
  FSYNC (pin 10) ──► GND
  REGOUT (pin 20) ──► 0.1µF ──► GND
  CPOUT (pin 18)  ──► 10nF ──► GND (充電泵電容)
```

### 5. USB-C 連接 (ESP32-C3 原生 USB)

```
                    USB-C Connector
              ┌────────────────────────┐
    VBUS ◄────┤ A4, B4    │  A5(CC1) ──┼── 5.1kΩ ──► GND
              │            │  B5(CC2) ──┼── 5.1kΩ ──► GND
GPIO19 ◄─────┤ A6(D+)    │  A7(D-) ──► GPIO20
              │            │  A1, A9,   │
  GND ◄──────┤ B1, B12   │  B9 ──────► GND
              └────────────────────────┘

  ESD Protection (選配):
              USBLC6-2SC6
    D+ ──┬──┬── D+
         │  │
         │  │  ┌─────┐
         └──┼──┤D+  GND├── GND
            └──┤D-  VCC├── NC
               └─────┘
    D- ──┬──┬── D-
         │  │
         │  │
```

### 6. 電池電壓監測分壓器（雙電源共用）

```
    BAT_RAIL (D1/D2 陰極共用點, 3.0~4.2V)
       │
       R1 = 100kΩ (1% 0603)
       │
       ├───────────── GPIO0 (ESP32-C3 ADC1_CH0)
       │
       R2 = 47kΩ (1% 0603)
       │
      GND

  旁路電容：GPIO0 ── 0.1µF ── GND (濾波)

  量測範圍：
    電池      V_BAT    V_ADC
    充滿      4.20V    1.34V
    標稱      3.70V    1.18V
    LDO 下限  3.05V    0.97V  ← 建議軟體低電量警告
    截止      2.80V    0.89V  ← ESP32 brownout

  ADC 轉換 (12-bit, 11dB attenuation)：
    v_adc = adc_raw × 3.3V / 4096
    v_bat = v_adc × (100k + 47k) / 47k = v_adc × 3.128
```

**註**：監測的是 OR-ing 後的共用節點，軟體無法區分當前是哪顆電池供電。若需區分，可在 D1 陰極/D2 陰極各接一組分壓到兩個 ADC 腳（GPIO0 + GPIO1），但多數場域使用場景不需要。
