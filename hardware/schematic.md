## 詳細電路

### 1. 電源電路（單一外接 16500 電池，獨立充電）

```
                         External 16500
                         (3.7V / 800mAh)
                              │
                         [Slide Switch]
                              │
                       ┌──────┴──────────┐
                       │   BAT_RAIL      ├──── BAT_ADC (GPIO0 分壓)
                       └──────┬──────────┘
                              │
                     ┌────────┴────────┐
                     │ ESP32-C3        │
                     │ Super Mini      │
                     │ (內建 LDO)      │
                     └────────┬────────┘
                              │
                           3.3V rail
                              │
                         10µF  0.1µF
                              │      │
                             GND    GND
```

**供電邏輯**：

| 狀況 | 電源來源 |
|------|---------|
| 開關 ON + 電池有電 | 16500 → ESP32-C3 Super Mini 內建 LDO → 3.3V 正常供電 |
| 開關 OFF | 斷電 |
| 電池沒電 | 系統關閉（更換電池後恢復） |

**注意事項**：
- 電池獨立充電（使用外部充電器），不經由板載電路
- USB-C 僅供程式燒錄，VBUS 不連接
- ESP32-C3 Super Mini 已內建 LDO 穩壓器，無需外接 ME6211 或 SS12 二極體

### 2. I2C 匯流排

```
+3.3V
  │
  ├─── 4.7kΩ ────┬──── SDA (ESP32-C3 GPIO6)
  │               │
  │               └──── SDA (MPU6500 pin 24)
  │
  ├─── 4.7kΩ ────┬──── SCL (ESP32-C3 GPIO7)
  │               │
  │               └──── SCL (MPU6500 pin 23)
  │
  └─────────────────── VCC (MPU6500 pin 8)
                      AD0 (MPU6500 pin 9) ──► GND

MPU6500 其餘腳位：
  INT (pin 12)  ──► NC (保留)
  CLKIN (pin 13) ──► GND
  FSYNC (pin 10) ──► GND
  REGOUT (pin 20) ──► 0.1µF ──► GND
  CPOUT (pin 18)  ──► 10nF ──► GND (充電泵電容)
```

### 3. 重置與下載電路

```
            +3.3V
              │
             10kΩ
              │
RST BTN ──┬──┴────── EN (ESP32-C3 pin 2)
          │
          │
         0.1µF
          │
         GND

            +3.3V
              │
             10kΩ (內部 pull-up, 可省略外接)
              │
BOOT BTN ────┴────── GPIO9 (ESP32-C3 pin 10)
          │
          │
         GND
```

### 4. USB-C 連接 (ESP32-C3 原生 USB)

```
                    USB-C Connector
              ┌────────────────────────┐
    VBUS ◄────┤ A4, B4    │  A5(CC1) ──┼── 5.1kΩ ──► GND
              │            │  B5(CC2) ──┼── 5.1kΩ ──► GND
GPIO19 ◄─────┤ A6(D+)    │  A7(D-) ──► GPIO20
              │            │  A1, A9,   │
  GND ◄──────┤ B1, B12   │  B9 ──────► GND
              └────────────────────────┘

  ESD Protection (選配):
              USBLC6-2SC6
    D+ ──┬──┬── D+
         │  │
         │  │  ┌─────┐
         └──┼──┤D+  GND├── GND
            └──┤D-  VCC├── NC
               └─────┘
    D- ──┬──┬── D-
         │  │
         │  │
```

### 5. 電池電壓監測分壓器（單一 16500 電池）

```
    BAT_RAIL (3.0~4.2V)
       │
       R1 = 100kΩ (1% 0603)
       │
       ├───────────── GPIO0 (ESP32-C3 ADC1_CH0)
       │
       R2 = 47kΩ (1% 0603)
       │
      GND

  旁路電容：GPIO0 ── 0.1µF ── GND (濾波)

  量測範圍：
    電池      V_BAT    V_ADC
    充滿      4.20V    1.34V
    標稱      3.70V    1.18V
    低電量    3.05V    0.97V  ← 建議軟體低電量警告
    截止      2.50V    0.80V  ← ESP32 brownout

  ADC 轉換 (12-bit, 11dB attenuation)：
    v_adc = adc_raw × 3.3V / 4096
    v_bat = v_adc × (100k + 47k) / 47k = v_adc × 3.128
```

**註**：本設計僅有單一外接 16500 電池（見「1. 電源電路」），BAT_RAIL 即電池正極經開關後電壓，無二極體壓降。
