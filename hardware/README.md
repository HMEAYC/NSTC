### 2b. MPU6500 連接

| MPU6500 腳位 | 連接 | 說明 |
|-------------|------|------|
| VCC | 3.3V | 供電 (2.375V~3.46V) |
| GND | GND | 共地 |
| SCL | GPIO7 + 4.7kΩ pull-up to 3.3V | I2C 時脈 |
| SDA | GPIO6 + 4.7kΩ pull-up to 3.3V | I2C 資料 |
| AD0 | GND | I2C 地址 = 0x68 |
| INT | (NC, 保留) | 中斷輸出 (暫不使用) |
| CLKIN | GND (或 NC) | 外部時脈 (不使用) |

**I2C pull-up 計算**：
- 4.7kΩ @ 3.3V → I²C 匯流排電容約 50pF → 上升時間約 260ns
- 符合 400kHz (Fast Mode) 規範 (< 300ns)

### 2d. 電池電壓監測

```
LiPo + (3.0V~4.2V)
     │
     R1 = 100kΩ (1%)
     │
     ├───── GPIO0 (ADC1_CH0, ESP32-C3)
     │
     R2 = 47kΩ (1%)
     │
    GND

V_ADC = V_BAT × R2 / (R1 + R2) = V_BAT × 47/147

量測範圍：
  V_BAT = 3.0V → V_ADC = 0.96V (放電截止)
  V_BAT = 3.7V → V_ADC = 1.18V (標稱電壓)
  V_BAT = 4.2V → V_ADC = 1.34V (充飽)

ESP32-C3 ADC 參考電壓 = 3.3V → 安全範圍內

ADC 轉換公式 (ESP32-C3, 12-bit, 衰減 11dB)：
  V_ADC = adc_raw × 3.3V / 4096
  V_BAT = V_ADC × (R1 + R2) / R2 = V_ADC × 147 / 47
```

USB-C Connector

│Pin │ Signal        │ Connection   │
├────┼───────────────┼──────────────┤
│A1  │ GND           │ GND          │
│A4  │ VBUS          │ TP4056 IN    │
│A6  │ D+            │ GPIO19       │
│A7  │ D-            │ GPIO20       │
│A9  │ GND           │ GND          │
│A12 │ SBU1          │ NC           │
│B4  │ VBUS          │ TP4056 IN    │
│B5  │ CC2           │ 5.1kΩ → GND │
│B9  │ GND           │ GND          │
│B12 │ SBU2          │ NC           │
│Others │ NC         │ NC           │
└────┴───────────────┴──────────────┘

CC1 (A5) 和 CC2 (B5) 各需要 5.1kΩ pulldown 到 GND
→ 告知 USB 主機為「無源裝置」以獲得 5V 供電

ES32-C3 內建 USB Serial/JTAG 控制器
→ 直接連接 D+/D- 即可，不需額外 USB-UART 晶片
```

### 3c. 天線注意事項 (ESP32-C3)

```

## 5. 編程與測試

### 5a. USB 下載

```
USB-C → ESP32-C3 原生 USB (GPIO19/20)
        ↓
無需外部 USB-UART 轉換晶片
        ↓
按住 BOOT (GPIO9→GND) → 接上 USB → 放開 BOOT
        ↓
idf.py flash monitor    ← VSCode 或命令列燒錄
```

### 5b. 初始上電測試步驟

1. **電源檢查**：接 USB 5V → 測 TP4056 BAT = 4.2V (無電池時)
2. **LDO 輸出**：測 ME6211 VOUT = 3.3V ± 2%
3. **模組啟動**：測 ESP32-C3 EN = 3.3V，串口出現 boot log
4. **I2C 掃描**：`i2cdetect` → 應看到 0x68 (MPU6500)
5. **IMU 讀取**：執行 main.c → 監控 accel/gyro 數據
6. **WiFi 連線**：設定 SSID → 看連接狀態
7. **WebSocket**：確認數據上傳到後端
8. **電池測試**：斷 USB → 確認電池供電正常 → 監控 ADC 電壓下降
