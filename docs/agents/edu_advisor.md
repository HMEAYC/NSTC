# edu_advisor — 教育建議生成器

## 角色定位
教育翻譯官：將分析數據轉譯為教師和家長能理解的建議與報告。

---

## 核心輸出模組

### 1. 固定骨架報告（`render_edu_markdown`）
- 報告標頭、系統設定、班級回饋
- 關注名單區塊、教學策略表、家長聯絡簿草稿

### 2. 教學策略調整建議
| 數據發現 | 策略建議 |
|---------|---------|
| 隊形崩潰 | 在該時間點前加入預告或視覺提示 |
| 熱區過度集中 | 設計「邊界探索遊戲」 |
| 互動密度過高 | 增加個人空間意識活動 |
| 同步誤差大 | 降低 BPM 或選節奏明確曲目 |

### 3. 可選 AI 擴寫（`augment_edu_report`）
- 有 API Key + `openai` 套件時附加 `## 五、AI 教學補充建議`
- 失敗時靜默回退，不中斷主流程

---

## 輸出格式
完整 Markdown 範例請見 `docs/agents/edu_advisor.md`（本檔案後半）。

## 輸出檔案
- `tmp/kinder-report.md` → `reports/YYYY-MM-DD-kinder-report.md`

---

## 限制
- 模板化建議，非診斷結論。
