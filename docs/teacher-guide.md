# 教師操作手冊

> 適用角色：`teacher`（教師）

---

## 1. 登入

**路徑：** `/dashboard/login`

- 輸入 Email + Password 登入
- 首次登入後會進入首頁

---

首頁顯示：

- **教案模板** — 建立可重複使用的課程階段模板

**📊 課程紀錄：**
- **課程紀錄** — 瀏覽過去所有課程與分析紀錄

**⚙️ 系統管理：**
- **班級管理** — 管理班級、教師與幼兒資料
- **裝置管理** — 檢視連線裝置與狀態

---

## 3. 課程管理

**路徑：** `/dashboard/sessions`

### 3.1 建立課程

1. 點擊「新增課程」
2. 填寫資訊：
   - **課程名稱**（必填）
   - **班級**（選填）— 選擇要上課的班級
   - **教案模板**（選填）— 選擇現有模板
   - **排程時間**（選填）— 設定上課時間
3. 點擊「儲存」→ 課程狀態為 `draft`

### 3.2 開課

1. 進入課程詳情頁（點擊課程列表中的課程）
2. 點擊「開始上課」
3. 狀態變為 `active`，開始時間自動記錄
4. ESP32 會自動連線到該課程的 WebSocket

### 3.3 指派裝置

**在課程詳情頁：**

1. 點擊幼兒列 → 配對彈窗
2. 選擇要配對的裝置
3. 點擊「手動配對」

**或使用自動配對：**

1. 點擊「🔗 跨模態自動配對」
2. 系統自動分析 IMU 資料 + 攝影機資料
3. 顯示配對建議 + 信心分數
4. 確認或手動調整

### 3.4 結束課程

1. 點擊「結束課程」
2. 狀態變為 `completed`，結束時間自動記錄
3. 系統自動計算評估指標

### 3.5 查看報告

1. 進入課程詳情頁
2. 點擊「查看報告」
3. 顯示：
   - 課程統計（IMU 資料量、裝置數）
   - 各幼兒評分
   - AI 分析報告

---

## 4. 教案模板

**路徑：** `/dashboard/templates`

### 4.1 建立模板

1. 點擊「新增模板」
2. 填寫資訊：
   - **模板名稱**（必填）
   - **描述**（選填）
   - **時長**（分鐘，選填）
3. 設計階段：
   - 點擊「新增階段」
   - 填寫：活動名稱、時長、音樂元素
   - 可新增多個階段
4. 點擊「儲存」

### 4.2 使用模板

- 建立課程時選擇模板
- 課程會自動載入模板的階段設計
- 課程進行中可切換階段

### 4.3 編輯模板

1. 進入模板詳情頁
2. 修改階段設計
3. 點擊「儲存」

## 5. 即時監控

**路徑：** `/dashboard/live/:sessionId`

### 5.1 功能

- **WS 連線狀態**：綠色 = 連線中、黃色 = 重連中、紅色 = 離線
- **6 軸即時數值**：AX, AY, AZ, GX, GY, GZ
- **加速度折線圖**：即時顯示動作強度
- **角速度折線圖**：即時顯示旋轉速度

### 5.2 攝影機功能

1. 點擊「開啟攝影機」
2. 瀏覽器請求攝影機授權
3. 自動開始 10fps JPEG 傳送
4. 後端 YOLO + MediaPipe 即時分析
5. 顯示：
   - 姿勢骨架疊加
   - CV 指標（團體投入度、隊形穩定度等）

### 5.3 操作流程

1. ESP32 開機後自動連線到課程的 WebSocket
2. 開啟 `/dashboard/live/<session_id>` 查看即時資料
3. 如裝置未被指派，會連到 `/ws/default`，可用 `/dashboard/live/default` 檢視

## 7. 評估指標

**路徑：** `/dashboard/assessment/:sessionId`

### 7.1 即時 IMU 指標（前端計算）

| 指標 | 說明 | 燈號 |
|------|------|------|
| 🏃 動作活躍度 | 加速度 RMS | 低 < 0.3g / 中 0.3~0.8g / 高 > 0.8g |
| 🎯 動作平穩度 | 加速度變異係數 | 平穩 < 0.3 / 普通 0.3~0.6 / 僵硬 > 0.6 |
| ⚖️ 身體穩定指數 | 1 − CV | 穩定 ≥ 0.7 / 尚可 ≥ 0.4 / 不穩 < 0.4 |

### 7.2 後端分析指標（需音樂參考）

| 指標 | 說明 |
|------|------|
| 🎵 節奏同步誤差 | 動作 vs 音樂拍點時間差 |
| 🧊 凍結反應/穩定度 | 音樂停止瞬間反應時間 + 穩定度 |

### 7.3 燈號系統

| 分數區間 | 燈號 | 意義 |
|---------|------|------|
| ≥ 0.85 | 🟢 極佳 | 表現優異 |
| ≥ 0.70 | 🟡 良好 | 正常發展 |
| < 0.70 | 🔴 需關注 | 建議教師介入 |

## 8. 班級管理

**路徑：** `/dashboard/classes`

### 8.1 查看班級

- 顯示所有班級列表
- 點擊班級 → 進入班級詳情頁

### 8.2 管理幼兒

**在班級詳情頁：**

- 新增幼兒：點擊「+ 註冊學員」→ 填寫姓名、學號、備註
- 編輯幼兒：點擊幼兒 → 修改資料
- 刪除幼兒：點擊幼兒 → 刪除

---

## 9. 裝置管理

**路徑：** `/dashboard/devices`

### 9.1 查看裝置

- 顯示所有已註冊的 ESP32 腰帶
- 欄位：名稱、Device ID、韌體版本、WiFi、電量、狀態

### 9.2 WiFi 設定

1. 點擊裝置 → 彈窗底部「📶 WiFi 設定」
2. 輸入新 WiFi SSID + 密碼
3. 儲存 → ESP32 自動重連

> 若 ESP32 偵測不到已知 WiFi，會自動建立 `HMEAYC-Setup` 熱點，手機/電腦連線後即可設定 WiFi。

---

## 10. 疑難排解

| 問題 | 解決方案 |
|------|----------|
| 裝置離線 | 檢查 WiFi 連線、電量、重啟裝置 |
| 課程無法開始 | 確認已指派裝置、班級已建立 |
| 即時監控無資料 | 確認 WS 連線狀態、裝置已開機 |
| 報告無資料 | 確認課程已結束、裝置有資料上傳 |
| 節奏同步無資料 | 需上傳音樂檔或設定 BPM |
