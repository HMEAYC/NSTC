# 管理員操作手冊

> 適用角色：`super_admin`（系統管理員）、`org_admin`（機構管理員）

---

## 1. 登入與帳號

| 功能 | 路徑 | 說明 |
|------|------|------|
| 登入 | `/dashboard/login` | Email + Password 登入 |
| 註冊 | `/dashboard/register` | 建立新帳號（需 org_code） |
| 接受邀請 | `/dashboard/accept-invite` | 完成 org invite 流程 |

**角色權限差異：**

| 功能 | super_admin | org_admin |
|------|-------------|-----------|
| 管理多個組織 | ✅ | ❌ |
| 切換組織（AdminOrgs） | ✅ | ❌ |
| 邀請使用者 | ✅ | ✅ |
| 編輯使用者 | ✅ | ✅ |
| 管理班級/幼兒/裝置 | ✅ | ✅ |

---

## 2. 帳號管理

**路徑：** `/dashboard/admin/users`

### 2.1 邀請使用者

1. 點擊「邀請使用者」
2. 輸入 Email + 選擇角色（teacher / parent / org_admin）
3. 系統發送邀請 Email
4. 受邀者點擊連結 → 設定密碼 → 完成註冊

### 2.2 管理帳號

- 列表顯示：Email、顯示名稱、角色、狀態
- 可停用/啟用帳號
- 可編輯顯示名稱

### 2.3 家長帳號

- 家長帳號由管理員建立或透過邀請流程
- 家長只能查看自己綁定的幼兒
- 管理員可在班級詳情頁綁定/解綁家長與幼兒

---

## 3. 班級管理

**路徑：** `/dashboard/classes`

### 3.1 建立班級

1. 點擊「新增班級」
2. 輸入班級名稱（必填）、年級（選填）
3. 儲存

### 3.2 管理班級

- 點擊班級 → 進入班級詳情頁
- 可編輯班級名稱、刪除班級
- 可新增幼兒到班級

### 3.3 幼兒管理

**在班級詳情頁：**

1. 點擊「+ 註冊學員」
2. 填寫姓名（必填）、學號（選填）、備註（選填）
3. 儲存

### 3.4 家長綁定

**在班級詳情頁：**

1. 選擇幼兒
2. 點擊「綁定家長」
3. 選擇家長帳號
4. 確認綁定

**解除綁定：**

1. 選擇幼兒
2. 點擊「解除綁定」
3. 確認

---

## 4. 裝置管理

**路徑：** `/dashboard/devices`

### 4.1 裝置列表

顯示所有已註冊的 ESP32 腰帶：

| 欄位 | 說明 |
|------|------|
| 名稱 | 裝置顯示名稱 |
| Device ID | MAC 地址 |
| 韌體版本 | 當前韌體版本 |
| WiFi SSID | 連線的 WiFi 名稱 |
| 訊強度 | WiFi 訊號強度 (RSSI) |
| IP | IP 位址 |
| 電量 | 電池電量百分比 |
| 最後上線 | 最後通話時間 |
| 狀態 | 🟢 連線中 / ⚪ 離線 |

### 4.2 WiFi 設定

1. 點擊裝置 → 彈窗底部「📶 WiFi 設定」
2. 輸入新 WiFi SSID + 密碼
3. 儲存 → ESP32 自動重連

**SoftAP Captive Portal 配網：**

當 ESP32 偵測不到已知 WiFi 時，會自動建立 `HMEAYC-Setup` 熱點（含 DNS 伺服器自動導向）：

1. 手機/電腦連線到 WiFi `HMEAYC-Setup`（無密碼）
2. DNS 自動解析所有域名到 `192.168.4.1`，Captive Portal 自動跳出
3. 或手動開啟 `http://192.168.4.1`
4. 選擇目標 WiFi → 輸入密碼 → 儲存
5. ESP32 自動重啟並連線到新 WiFi

### 4.3 網路掃描（super_admin）

1. 開啟裝置管理頁面
2. 點擊「掃描網路」按鈕
3. 系統透過 ARP 表掃描區域網路，過濾 Espressif OUI 前綴
4. 顯示未註冊的 ESP32 裝置
5. 點擊「註冊」即可加入裝置列表

### 4.3 韌體更新

**路徑：** `/dashboard/firmware`

1. 上傳新韌體 binary + 版本號
2. ESP32 每 24 小時自動檢查更新
3. 發現新版後自動下載 + 重啟

## 5. 課程管理

**路徑：** `/dashboard/sessions`

### 5.1 建立課程

1. 點擊「新增課程」
2. 填寫：課程名稱（必填）、班級（選填）、教案模板（選填）、排程時間（選填）
3. 儲存 → 課程狀態為 `draft`

### 5.2 開課

1. 進入課程詳情頁
2. 點擊「開始上課」
3. 狀態變為 `active`，開始時間自動記錄

### 5.3 指派裝置

**在課程詳情頁：**

1. 點擊幼兒列 → 配對彈窗
2. 選擇裝置 → 手動配對
3. 或使用「🔗 跨模態自動配對」（需攝影機資料）

### 5.4 結束課程

1. 點擊「結束課程」
2. 狀態變為 `completed`，結束時間自動記錄
3. 系統自動計算評估指標

### 5.5 查看報告

1. 進入課程詳情頁
2. 點擊「查看報告」
3. 顯示課程統計、各幼兒評分、AI 分析報告

---

## 6. 教案模板

**路徑：** `/dashboard/templates`

### 6.1 建立模板

1. 點擊「新增模板」
2. 填寫：模板名稱、描述、時長（分鐘）
3. 設計階段（每個階段包含：活動名稱、時長、音樂元素）
4. 儲存

### 6.2 使用模板

- 建立課程時選擇模板
- 課程會自動載入模板的階段設計
- 課程進行中可切換階段

---

## 7. 即時監控

**路徑：** `/dashboard/live/:sessionId`

### 7.1 功能

- WS 連線狀態指示燈（綠/黃/紅）
- 6 軸即時數值卡片（AX, AY, AZ, GX, GY, GZ）
- 加速度（g）即時折線圖
- 角速度（dps）即時折線圖
- 攝影機預覽（可選）

### 7.2 攝影機功能

1. 點擊「開啟攝影機」
2. 瀏覽器請求攝影機授權
3. 自動開始 10fps JPEG 傳送
4. 後端 YOLO + MediaPipe 即時分析
5. 顯示骨架疊加 + CV 指標

## 8. 系統管理（super_admin 專用）

### 8.1 組織管理

**路徑：** `/dashboard/admin`

- 列出所有組織
- 可新增、編輯、刪除組織
- 切換組織後，所有頁面資料自動更新

### 8.2 資料匯出

**合規匯出（IRB / 審計）：**

```bash
# 匿名化資料匯出
curl -X POST http://localhost:8000/api/admin/export/anonymized \
  -H "Content-Type: application/json" \
  -d '{"format":"json"}'

# 同意書報表
curl http://localhost:8000/api/admin/export/consent-report
```

### 8.3 審計日誌

```bash
curl http://localhost:8000/api/admin/audit-logs
```

## 9. 疑難排解

| 問題 | 解決方案 |
|------|----------|
| 裝置離線 | 檢查 WiFi 連線、電量、重啟裝置 |
| WiFi 設定失敗 | 使用 SoftAP Captive Portal 或 USB 重新燒錄 |
| 課程無法開始 | 確認已指派裝置、班級已建立 |
| 報告無資料 | 確認課程已結束、裝置有資料上傳 |
| 家長看不到幼兒 | 確認已綁定家長帳號 |
